AltStats provides an easy-to-read summary of visitor activity on your Wordpress site.

== Description ==

AltStats is an embarrassingly easy to install statistics
plugin for wordpress that shows live statistics about 
referer data, content hit count and browser versions.

AltStats is not meant to be a full featured log analyzer. 
It does all the analysis on demand and shows them on an easily accessible single page.

== Installation ==

1. Upload `altstats.php` and the `altstats` directory to your plugins folder, usually `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.


== Frequently Asked Questions ==

= Where do I see the statistics? =

Dashboard -> AltStats

= Where do I configure this? =

This plugin requires no configuration.
